Lesson 1: Introduction to HTML

HTML (HyperText Markup Language) is the backbone of all web pages. It provides structure and meaning to content.

ðŸ“Œ Basic HTML Structure
Code Example
<!DOCTYPE html> <html> <head> <title>My First Page</title> </head> <body> <h1>Hello, HTML!</h1> <p>This is my first webpage.</p> </body> </html>
Explanation

<!DOCTYPE html> defines the document type. The <html> tag wraps the entire page. Inside it, we use <head> for metadata (like the title) and <body> for visible content.

Lesson 2: Basic Tags

ðŸ“Œ Headings & Paragraphs
Code Example
<h1>Main Heading</h1> <h2>Subheading</h2> <p>This is a paragraph.</p>
Explanation

Headings are defined using <h1> to <h6>. The <p> tag is used for normal text paragraphs.

ðŸ“Œ Text Formatting
Code Example
<p>This is <b>bold</b> and <i>italic</i> text.</p> <br> <p>New line after break.</p>
Explanation

<b> makes text bold, <i> italicizes text, and <br> forces a line break.

Lesson 3: Links & Images

ðŸ“Œ Hyperlinks
Code Example
<a href="https://example.com">Visit Example</a>
Explanation

The <a> tag creates links. The href attribute defines the URL where the link points.

ðŸ“Œ Images
Code Example
<img src="image.jpg" alt="My Image" width="200">
Explanation

<img> inserts an image. Use src for the file path, alt for accessibility, and width or height for resizing.

Lesson 4: Lists & Tables

ðŸ“Œ Lists
Code Example
<ul> <li>Apple</li> <li>Banana</li> </ul> <ol> <li>First</li> <li>Second</li> </ol>
Explanation

<ul> creates an unordered (bulleted) list. <ol> creates an ordered (numbered) list. Each item is wrapped with <li>.

ðŸ“Œ Tables
Code Example
<table border="1"> <tr><th>Name</th><th>Age</th></tr> <tr><td>Alice</td><td>22</td></tr> <tr><td>Bob</td><td>25</td></tr> </table>
Explanation

<table> defines a table. Rows use <tr>, headers use <th>, and data cells use <td>.

Lesson 5: Forms

ðŸ“Œ Basic Form
Code Example
<form> Name: <input type="text" name="username"><br> Email: <input type="email" name="email"><br> <input type="submit" value="Submit"> </form>
Explanation

Forms allow user input. <input type="text"> collects text, type="email" ensures valid email format, and type="submit" adds a button.

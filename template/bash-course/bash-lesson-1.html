🧠 What is Bash?

Bash (short for Bourne Again SHell) is a powerful command-line tool that lets you interact with your computer by typing commands instead of clicking around.

It’s commonly used in Linux, macOS, WSL (Windows Subsystem for Linux), and on servers around the world.

Think of Bash as your way to talk directly to your operating system — like texting your computer what you want it to do.

💡 Why Should You Learn Bash?

  • 🚀 Speed: Do tasks in seconds that might take minutes using a mouse.
  • 🤖 Automation: Automate repetitive tasks using scripts.
  • 🖥️ Server Control: Most servers don't have a graphical interface — only a terminal.
  • 🎯 Precision: Use exact commands with fine-tuned options.
  • 📂 Control: Create, delete, move files/folders without leaving your keyboard.

💬 Understanding the Command Prompt

When you open a terminal, you’ll see something like:

student@pc:~$

Here’s what that means:

  • student — your username
  • pc — the name of the computer (host)
  • ~ — your current folder (home directory)
  • $ — the shell is ready for your command (use # if you're root user)

Let’s try a few basic commands:

student@pc:~$ pwd /home/student student@pc:~$ date Mon Aug 19 13:45:12 UTC 2025

📚 How to Get Help

If you don’t know how to use a command, you can usually ask it directly for help:

$ echo --help Usage: echo [SHORT-OPTIONS] [STRING]... Print the STRING(s) to standard output.

Or you can use the manual:

$ man echo

Use q to quit the manual view when you're done reading it.

🎯 Summary

  • Bash is a shell used to communicate with your OS using commands.
  • You can navigate folders, manage files, and automate tasks.
  • It’s fast, powerful, and used by professionals worldwide.

✅ Try this:

$ echo "I'm ready to learn Bash!" I'm ready to learn Bash!

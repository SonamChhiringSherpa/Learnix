Redirecting Output

In Bash, you can control where command output goes using redirection operators:

  • > – Overwrites the file with new output
  • >> – Appends output to the end of the file
$ echo "Hello" > notes.txt $ cat notes.txt Hello $ echo "World" >> notes.txt $ cat notes.txt Hello World

Viewing File Content

Use these commands to preview the beginning or end of a file:

  • head -n [number] filename – shows the first lines
  • tail -n [number] filename – shows the last lines
$ head -n 2 notes.txt Hello World $ tail -n 1 notes.txt World

Searching in Files

To search for lines containing a specific word or pattern, use grep:

$ grep "Hello" notes.txt Hello

Try it Yourself

Here's a mini challenge for practice:

  • Create a file named log.txt and add several lines to it.
  • Use head and tail to view specific parts.
  • Search for a word in your file using grep.

In today's digital world, knowing how to code is more than just a useful skill—it's a superpower. Whether you're building websites, automating tasks, or just trying to understand how the tools around you work, programming opens up a world of possibilities. That's exactly why I decided to create this learning website: a simple, beginner-friendly platform to help others get started with the core building blocks of web development and scripting

As someone who has spent countless hours digging through tutorials, watching videos, and testing code in different environments, I realized how overwhelming the learning journey can be—especially when the resources are scattered, overly complex, or not beginner-focused. I wanted to build something different. This site is designed to be a clear, hands-on, and distraction-free space where learners can dive straight into coding, see real-time results, and build confidence as they go.

Feature Overiview

This learning website is built to help beginners learn HTML, CSS, JavaScript, and Bash in a clear, structured, and interactive way. Instead of just reading code, users can experiment with it, see real-time results, and build a deeper understanding of how things work. Below is an overview of the main features that make this platform both practical and engaging:

Live Preview Output

As users type their code, a live preview pane updates in real time.

Progressive Learning

Each topic builds on what came before, making the learning curve smooth and logical.

Teck Stack used

User Experience (UX) & UI Design

A learning platform isn't just about delivering content—it's about creating an environment where users can focus, explore, and enjoy the process of learning. When designing this site, I made user experience and interface clarity a top priority, especially considering that most users might be beginners stepping into code for the first time.

01

Simple and Intuitive Navigation

Each section follows a logical flow, guiding learners from the basics to more advanced concepts step by step.

02

Bold Graphics

Design Philosophy

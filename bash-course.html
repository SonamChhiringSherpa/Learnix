Lesson 1: Introduction to Bash

Bash is a Unix shell and command language. You can run commands and automate tasks.

Terminal (read-only)
$ echo "Hello, Bash!" Hello, Bash! $ uname -s Linux
Explanation

echo prints text. uname -s shows the operating system kernel.

Lesson 2: Creating Directories

Terminal (read-only)
$ mkdir projects $ ls -l drwxr-xr-x 2 user user 4096 Jan 1 00:00 projects $ cd projects $ pwd /home/user/projects
Explanation

mkdir creates directories. Use ls -l to list details and pwd to show the current path.

Lesson 3: Viewing & Editing Files

Terminal (read-only)
$ echo "Hello" > notes.txt $ cat notes.txt Hello $ echo "World" >> notes.txt $ cat notes.txt Hello World
Explanation

Use > to create/overwrite a file and >> to append. cat prints file content.

Lesson 4: Permissions

Terminal (read-only)
$ touch script.sh $ chmod +x script.sh $ ls -l script.sh -rwxr-xr-x 1 user user 0 Jan 1 00:00 script.sh
Explanation

chmod +x makes a file executable. The x flag allows running a script.

Lesson 5: Scripting Basics

Terminal (read-only)
$ cat hello.sh #!/usr/bin/env bash name=$1 echo "Hello, $name!" $ bash hello.sh Learner Hello, Learner!
Explanation

This script reads an argument and prints a greeting. Pass arguments as $1, $2, etc.

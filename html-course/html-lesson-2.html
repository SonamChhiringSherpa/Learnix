Core Building Blocks

This lesson covers the most-used tags: headings, paragraphs, emphasis, line breaks, and comments. Mastering these lets you structure readable content quickly.

You’ll also meet <div> and <span> — generic containers used for grouping content and inline text respectively.

Tag Reference & Tips

  • <h1>...<h6> — Heading hierarchy. Use exactly one <h1> per page.
  • <p> — Paragraph block for text.
  • <strong> — Strong importance (often bold).
  • <em> — Emphasis (often italic).
  • <br> — Line break (use sparingly).
  • <div> — Block-level container.
  • <span> — Inline container.

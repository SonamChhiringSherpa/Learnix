Lesson 4: Lists & Tables

In HTML, lists are used to group related items, while tables are used to organize data in rows and columns. Lists are great for menus, steps, or grouping items, while tables are useful for structured data like schedules or reports.

Unordered Lists (<ul>)

Unordered lists display items with bullet points. They are used when the order of items doesn’t matter.

Ordered Lists (<ol>)

Ordered lists display items in a numbered format. They are used when the order of items is important, such as steps in instructions.

Nested Lists

You can place lists inside other lists. This is useful for creating sub-menus or categories.

Basic Table

Tables in HTML are created using the <table> element. Rows are defined with <tr>, headers with <th>, and data cells with <td>.

Table with rowspan and colspan

The rowspan and colspan attributes allow a cell to span multiple rows or columns, making your table more flexible.

Table with Styling

You can style tables with inline CSS or external stylesheets to improve readability.

Understanding Lists & Tables

Lists and tables are essential for organizing content on a webpage. Lists help display items in sequence or bullet form, while tables present structured data using rows and columns.

  • <ul>...</ul> → Creates an unordered list (bulleted).
  • <ol>...</ol> → Creates an ordered list (numbered).
  • <li>...</li> → Defines an item in a list.
  • <table>...</table> → Creates a table structure.
  • <tr>...</tr> → Defines a table row.
  • <th>...</th> → Defines a table header cell (bold and centered).
  • <td>...</td> → Defines a table data cell (normal cell).

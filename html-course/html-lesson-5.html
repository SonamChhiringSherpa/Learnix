Lesson 5: Forms & Inputs

Forms are used to collect user input. They include fields like text boxes, checkboxes, radio buttons, and buttons to make a website interactive and functional.

Understanding Forms & Inputs

Forms are made up of different input elements, each serving a unique purpose. Here are the key tags and their functions:

  • <form>...</form> → Defines a form container.
  • <input type="text"> → Single-line text input field.
  • <input type="email"> → Email input with validation.
  • <input type="password"> → Password (hidden characters).
  • <input type="number"> → Numeric input with min/max.
  • <input type="checkbox"> → Checkbox for multiple choices.
  • <input type="radio"> → Radio button for single choice.
  • <select>...</select> → Dropdown list.
  • <textarea>...</textarea> → Multi-line text input.
  • <button> → Buttons (submit, reset, custom actions).

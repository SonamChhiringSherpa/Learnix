What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages. It provides the basic structure of a page, which is enhanced and styled using CSS, and made interactive with JavaScript.

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>Welcome to HTML learning.</p>
  </body>
</html>

HTML Page Structure

An HTML page is made up of elements wrapped in <tags>. The main parts are:

  • <!DOCTYPE html> – Declares this is an HTML5 document.
  • <html> – Root element of the page.
  • <head> – Contains metadata (title, styles, etc.).
  • <body> – Holds the visible page content.

First Output

When you open the above code in a browser, you’ll see:

Hello, World!


Welcome to HTML learning.

This is the simplest example of an HTML page in action.

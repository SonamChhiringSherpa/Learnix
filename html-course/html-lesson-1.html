What is HTML?

HTML (HyperText Markup Language) is the building block of the web. It describes the structure of web pages using elements (tags). Every website you visit is built on top of HTML.

Think of HTML as the foundation of a house — without it, nothing stands. CSS adds design (paint, layout), and JavaScript adds interactivity (doors that open when clicked).

Understanding the Structure

An HTML document is structured using different parts. Let’s break down the example:

  • <!DOCTYPE html> → Defines this is an HTML5 document.
  • <html>...</html> → Root element of the page.
  • <head>...</head> → Contains metadata like title, styles, links.
  • <title> → Sets the page’s title (shown in the browser tab).
  • <body>...</body> → Contains all the visible content of the page.
  • <h1> → A heading tag (largest).
  • <p> → A paragraph of text.
